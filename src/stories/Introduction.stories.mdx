# Council Data Project

Welcome to the [Council Data Project](https://councildataproject.github.io/) frontend component library.

Council Data Project is an open-source project dedicated to providing journalists, activists,
researchers, and all members of each community we serve with the tools they need to stay informed
and hold their Council Members accountable.

By combining and simplifying sources of information on Council meetings and actions, CDP ensures
that everyone is empowered to participate in local government.

Each municipality that CDP supports (a "_CDP Instance_") has open source maintainers which write code
to gather municipality meeting information and compile them into a single resource to then be
processed, stored, and made accessible by our general CDP tooling.

# CDP Frontend

This [repo](https://github.com/CouncilDataProject/cdp-frontend) / storybook contains all of the
components used to create the frontend of CDP instances deployed to each municipality. Contributions
here will be made available to all CDP instances. The components are written in TypeScript and React
and use the [Mozilla Protocol Design System](https://protocol.mozilla.org/) for styling.

# Storybook

This component library is also developed using Storybook. Below you can find documentation and
guides for getting up and running with Storybook.

Storybook helps you build UI components in isolation from an app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during
development, testing, or QA.
