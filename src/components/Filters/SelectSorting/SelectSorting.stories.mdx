# Select Sorting

### Display radio checkboxes to select sorting options

The width of SelectSorting takes on the width of its container.

<Props of={SelectSorting} />

#### Sort By Most relevant is the default sorting.

<Story name="Default sort by: Most relevant">
  <SelectSorting
    state={{
      by: "value",
      order: "desc",
      label: "Most relevant",
    }}
    update={action("update-sorting-state")}
    sortOptions={[
      { by: "value", order: "desc", label: "Most relevant" },
      { by: "date", order: "desc", label: "Newest first" },
      { by: "date", order: "asc", label: "Oldest first" },
    ]}
    onPopupClose={action("on-popup-close")}
  />
</Story>

### Playground

#### Experiment with these stories with Knobs addon in Canvas mode.

Change the values of the sort state to change what sort option is selected.

<Story name="Sort State Playground">
  <SelectSorting
    state={select(
      "Sort Results By",
      {
        "Most relevant": { by: "value", order: "desc", label: "Most relevant" },
        "Newest first": { by: "date", order: "desc", label: "Newest first" },
        "Oldest first": { by: "date", order: "asc", label: "Oldest first" },
      },
      { by: "value", order: "desc", label: "Most relevant" }
    )}
    update={action("update-sorting-state")}
    sortOptions={[
      { by: "value", order: "desc", label: "Most relevant" },
      { by: "date", order: "desc", label: "Newest first" },
      { by: "date", order: "asc", label: "Oldest first" },
    ]}
    onPopupClose={action("on-popup-close")}
  />
</Story>

Change the values of sort options array to change what sort options are visible.

<Story name="Sort By Options Playground">
  <SelectSorting
    state={{
      by: "value",
      order: "desc",
      label: "Most relevant",
    }}
    update={action("update-sorting-state")}
    sortOptions={object("Sort Results By Options", [
      { by: "value", order: "desc", label: "Most relevant" },
      { by: "date", order: "desc", label: "Newest first" },
      { by: "date", order: "asc", label: "Oldest first" },
    ])}
    onPopupClose={action("on-popup-close")}
  />
</Story>
