# Select Sorting

### Display radio checkboxes to select sorting options

The width of SelectSorting takes on the width of its container.

<Props of={SelectSorting} />

#### Sort By Most relevant is the default sorting.

<Story name="Default sort by: Most relevant">
  <SelectSorting
    state={{
      by: "value",
    }}
    update={action("update-sorting-state")}
    sortByOptions={["date", "value"]}
  />
</Story>

### Playground

#### Experiment with these stories with Knobs addon in Canvas mode.

Change the values of the sort state to change what sort options are selected.

<Story name="Sort State Playground">
  <SelectSorting
    state={{
      by: select("Sort Results By", {
        "Most recent": "date",
        "Most relevant": "value",
      }),
    }}
    update={action("update-sorting-state")}
    sortByOptions={["date", "value"]}
  />
</Story>

Change the values of sort by options array to change what sort by options are visible.

<Story name="Sort By Options Playground">
  <SelectSorting
    state={{
      by: "",
    }}
    update={action("update-sorting-state")}
    sortByOptions={array("Sort Results By Options", ["date", "value"])}
  />
</Story>
